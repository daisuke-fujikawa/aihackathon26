# Product Overview

**Yo-i Facilitator（酔い・良いファシリテーター）** — 「AI × ビール × コミュニケーション」をコンセプトに、AIが酔っ払いキャラクター「ヨイさん」として飲み会のファシリテーションを行うChromeベースのWebアプリケーション。

## Core Capabilities

1. **音声認識＋発話量可視化**: マイクからの音声をリアルタイム認識し、発話量をビールジョッキUIで可視化
2. **酔っ払いAIファシリテーション**: 未発話者へのパス出し、沈黙キラー、「乾杯」演出、帰宅リマインド、休憩提案
3. **音声合成**: ヨイさんの酔っ払い風音声でスピーカーから応答再生
4. **コンテキスト検知**: 会話内容からキーワードを拾い、状況に応じた介入を自動実行

## Target Use Cases

- 飲み会・懇親会での発言の偏り解消と沈黙回避
- 切り上げタイミングの提案（終電リマインド）
- AIハッカソンのデモとしてインパクトのある体験を提供

## Value Proposition

AIがあえて「一番の酔っ払い」という低ステータスな役割を演じることで、参加者の心理的安全性を最大化。誰もが等身大で話せる場を作る。キャラクター「ヨイさん」は常にビール3杯目、陽気で全肯定、少し物忘れが激しい設定。
